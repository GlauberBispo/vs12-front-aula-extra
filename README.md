## ⚙️ Padrão de nomenclatura de repositório:

O nome do repositório será composto por:

> `edição do vem ser` + `trilha` + `modulo` + `número task`

Exemplo:

nome do repo: `vs12-front`

```
vs12-front/01-html-e-css/task-01
```
